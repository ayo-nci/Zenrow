Setup Instructions

To use this comparison tool, you need to set up a Node.js server that handles the scraping. Follow these steps to get started:

  1. Create a file named server.js with the provided backend code
  2. Install the required dependencies using npm
  3. Add your ZenRows API key to a .env file
  4. Start the server with node server.js
